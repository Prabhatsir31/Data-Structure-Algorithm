## 🚀 Getting Started



### 1. Clone the Repository
```bash
git clone https://github.com/Prabhatsir31/Data-Structure-Algorithm.git
cd Data-Structure-Algorithm
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App
```bash
npm start
```
- The app will be available at `http://localhost:3000/`.
